# ShopZone - MERN E-commerce Website

ShopZone is a fully responsive E-commerce web application built using the MERN Stack.
Users can browse products, search products, filter by category, sort products, view product details, and manage cart items.

## Features

* Responsive Navbar
* Product Listing
* Product Details Page
* Search Products
* Category Filter
* Sort Products
* Add to Cart
* Remove from Cart
* Quantity Management
* LocalStorage Cart Persistence
* Skeleton Loaders
* Responsive Mobile Design
* Login Page UI
* Signup Page UI

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Tailwind CSS
* React Icons
* React Hot Toast

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication
* bcryptjs

frontend/

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/shopzone.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
